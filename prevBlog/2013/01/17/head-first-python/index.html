P120

The with statement, when used with files, it negates the need to include a finally suite to handle the closing of a potentially opened file. The usual try/except/finally pattern:

try:
    data = open('its.txt', 'w')
    print("it's ...", file=data)
except IOError as err:
    print('File error: ' + str(err))
finally:
    if 'data' in locals():
        data.close()

The use of with negates the need for the finally suite.

try:
    with open('its.txt', 'w') as data:
        print("it's ...", file=data)
except IOError as err:
    print('File error: ' + str(err))
