### Data Model
Three components:

* Structures
* Constraints
* Operations

Releational Algebra has two semantics:

* Set sematics = standard Relational Algebra
  + Union
  + Intersection
  + Difference
  + Selection
  + Projection
  + Join
* Bag semantics = extended Relational Algebra
  + Duplication eliminaiton
  + Grouping and aggregation
  + Sorting
